WORDLY DICTIONARY APP

Project Description

Wordly is a Single Page Application (SPA) that allows users to search for word definitions, pronunciation, examples, and synonyms using a dictionary API.

 Features

- Search for any English word
- View definitions and meanings
- See pronunciation and examples
- Display synonyms
- Error handling for invalid words
- Dynamic theme toggle (light/dark mode)

 Technologies Used

- HTML
- CSS
- JavaScript
- Dictionary API (https://api.dictionaryapi.dev)

Project Structure

- index.html → Structure of the app
- style.css → Styling
- script.js → Functionality

How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/wordly-app.git

2. Open the folder:
   cd wordly-app

3. Open index.html in your browser
